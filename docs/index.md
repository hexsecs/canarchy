# CANarchy Docs

CANarchy is a CLI-first CAN security research toolkit with structured outputs, protocol-aware workflows, and a shared core model for future CLI, shell, and TUI parity.

This docs site is the project documentation portal hosted from the same repository as the code.

Start here:

* Operators: [Project Overview](overview.md) and [Command Spec](command_spec.md)
* Developers: [Architecture](architecture.md), [Docs Workflow](docs_site.md), and [TUI Plan](tui_plan.md)
* Agents: [Agent Guide](agents.md)

Site goals:

* keep the CLI contract explicit and current
* keep code and docs versioned together
* make operator, developer, and agent workflows easy to find
